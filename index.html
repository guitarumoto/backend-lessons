Videoaulas de Desenvolvimento Backend

Aula 1: O papel do desenvolvimento Back-end no Desenvolvimento Web

Nesta aula, discutiremos como o back-end processa dados, gerencia a lógica de negócios e conecta o front-end aos bancos de dados. Você entenderá a importância de construir sistemas robustos e escaláveis utilizando tecnologias como Node.js e Express.

  • Papel do back-end no desenvolvimento web.
  • Visão geral do Node.js e do framework Express.
  • Criação de APIs RESTful com essas tecnologias.

Aula 2: Configuração do Ambiente de Desenvolvimento

Nesta aula, você configurará o ambiente de desenvolvimento instalando Node.js e npm, iniciando seu projeto com npm init e criando um servidor básico com Express. Esse setup é fundamental para iniciar qualquer projeto back-end.

  • Instalação do Node.js e npm.
  • Inicialização de um projeto Node.js com npm init.
  • Configuração básica de um servidor Express.

Aula 3: Introdução ao MySQL e ao Sequelize

Descubra como o MySQL funciona como um banco de dados relacional e como o Sequelize, um ORM poderoso, simplifica a interação entre a aplicação e o banco de dados, acelerando o desenvolvimento.

  • Características do banco de dados relacional MySQL.
  • O que é o ORM Sequelize e seus benefícios.
  • Configuração da conexão entre o Sequelize e o MySQL.

Aula 4: Modelagem de Dados Relacionais com Sequelize

Aprenda a estruturar os dados de sua aplicação usando o Sequelize. Você verá como criar modelos que representam entidades do sistema e definir relacionamentos entre elas, garantindo uma base de dados consistente.

  • Definição de modelos e mapeamento objeto-relacional.
  • Criação de modelos para Usuários, Publicações, Comentários e Curtidas.
  • Estabelecimento de relacionamentos (associações) entre os modelos.

Aula 5: Criação de APIs RESTful com Express

Nesta aula, você desenvolverá a capacidade de criar APIs RESTful utilizando o Express. Será abordada a definição de rotas e métodos HTTP, permitindo a comunicação eficaz entre o cliente e o servidor.

  • Princípios das APIs RESTful.
  • Definição e implementação de rotas e métodos HTTP (GET, POST, PUT, DELETE).
  • Estruturação de endpoints para os recursos da aplicação.

Aula 6: Validação e Sanitização de Dados no Servidor

Nesta aula, enfatizaremos a importância de validar e sanitizar os dados que chegam ao servidor. Técnicas e ferramentas serão apresentadas para garantir que somente informações seguras e corretamente formatadas sejam processadas.

  • Importância da validação de dados no back-end.
  • Uso de bibliotecas como express-validator.
  • Validação de campos específicos (e-mail único, senha com letras e números).

Aula 7: Cadastro de usuários

Nesta aula, você aprenderá como criar um sistema de cadastro de usuários robusto, envolvendo a criação de rotas para registro, verificação de e-mail duplicado e a criptografia de senhas para armazenamento seguro.

  • Criação de rotas para cadastro de novos usuários.
  • Verificação de unicidade do e-mail no banco de dados.
  • Criptografia de senhas com bcrypt para armazenamento seguro.

Aula 8: Autenticação com JSON Web Tokens (JWT)

Nesta aula, abordaremos os conceitos de autenticação utilizando JSON Web Tokens (JWT), mostrando como implementar um sistema de login seguro e gerar tokens para gerenciar sessões de usuário.

  • Conceitos de autenticação e JWT.
  • Implementação da funcionalidade de login.
  • Geração e validação de tokens JWT para sessões de usuário.

Aula 9: Proteção de Rotas e Middleware de Autenticação

Nesta aula, você aprenderá a proteger rotas sensíveis criando middlewares que validam tokens JWT, garantindo que somente usuários autenticados tenham acesso a determinadas funcionalidades.

  • Criação de middleware para verificar tokens JWT.
  • Proteção de rotas que exigem autenticação.
  • Controle de acesso baseado em papéis (administrador, usuário comum).

Aula 10: Atualização de Dados Pessoais do Usuário

Nesta aula, você verá como criar rotas que permitam aos usuários atualizar seus dados pessoais de forma segura, validando e sanitizando as informações para garantir integridade e segurança.

  • Rotas para que o usuário possa atualizar nome e senha.
  • Verificação de autenticação antes da atualização.
  • Validação e sanitização dos dados durante a atualização.

Aula 11: Upload de Imagens em Publicações

Nesta aula, você aprenderá como implementar o upload de imagens em publicações utilizando a biblioteca Multer. Serão abordadas as configurações de armazenamento e a validação dos arquivos enviados.

  • Implementação de upload de arquivos com Multer.
  • Configuração de armazenamento local ou em serviço externo.
  • Validação do tipo (MIME type) e tamanho dos arquivos enviados.

Aula 12: Listagem e Filtragem de Publicações

Nesta aula, você aprenderá a criar rotas para listar todas as publicações e aplicar filtros que possibilitem a busca por categorias ou faixas de preço, além de implementar paginação para melhorar a performance.

  • Rotas para listar todas as publicações disponíveis.
  • Implementação de filtros por critérios específicos (e.g., categorias, preço máximo).
  • Paginação e ordenação dos resultados para melhor performance.

Aula 13: Comentários e Curtidas em Publicações

Nesta aula, você aprenderá como estruturar modelos para comentários e curtidas, permitindo que usuários interajam com as publicações. Serão criadas rotas para adicionar e listar essas interações.

  • Estruturação de modelos para Comentários e Curtidas.
  • Rotas para adicionar comentários e curtidas sem necessidade de autenticação.
  • Rotas para listar comentários de uma publicação específica.

Aula 14: Gerenciamento de Permissões e Segurança

Nesta aula, você aprenderá a implementar mecanismos de segurança para garantir que os usuários só possam modificar seus próprios dados, protegendo informações sensíveis e garantindo a integridade do sistema.

  • Garantir que usuários não possam modificar dados de outros usuários.
  • Implementação de verificações de propriedade nas rotas de atualização e exclusão.
  • Medidas para proteger dados sensíveis.

Aula 15: Padronização de Respostas e Tratamento de Erros

Nesta aula, abordaremos como padronizar as respostas da API e implementar um middleware global para tratamento de erros, facilitando a manutenção e o diagnóstico de problemas na aplicação.

  • Criação de um middleware global para tratamento de erros.
  • Padronização das respostas de sucesso e erro da API.
  • Envio de mensagens de erro claras e significativas ao cliente.
